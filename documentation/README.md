## Tech Stack

- Django 5.2.3
- Django REST Framework
- SQLite (Development)
- JWT Authentication
- CORS support

## Features

- Complete Lawyer Management System
- Client Management
- Case Management
- Document Management
- Task Management
- REST API Backend
- Token-based Authentication
- Cross-Origin Resource Sharing (CORS) enabled
